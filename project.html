GLOBAL FINANCIAL CRISIS. 2007 - 2008

The global financial crisis started at the beginning of 2007 with a crisis in the US subprime mortgage market, and extended to the international banking system. The following dynamic chart exhibits key indicators in the period between March 2007 and December 2010. The green dots (left axis) correspond to the S&P 500 index which is a stock market index. The red dots (right axis) correspond to the VIX index which a (implied) volatility index constructed from option prices. This last index can be interpreted as a measure of financial risk. The chart dinamically shows the value of each index for each date. Additionally, the orange circles correspond to important events that ocurred during the period under analysis. As the user rolls the mouse over these circles a description of the event appears.
